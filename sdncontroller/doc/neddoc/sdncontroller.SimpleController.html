Simple Module SimpleController

Package: sdncontroller
File: src/SimpleController.ned

C++ definition

~SimpleController is heritaged from IApp Module works like SDN controller which supports OFPv100 protocol. You can edit, add your own applications Applications already done: App1,App2,App3,App4 (Descryption of apps are in api.cc)
